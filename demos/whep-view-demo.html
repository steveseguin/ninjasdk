WHEP View Demo

Watch streams from WHEP-compatible services

How to use:
  1. First, publish a stream using the WHIP Publish Demo or another WHIP source
  2. Enter the WHEP endpoint URL (same stream ID as WHIP, but with /whep/ instead of /whip/)
  3. Click "Connect" to start viewing

Meshcast example: If you published to https://cae1.meshcast.io/whip/MyStream, view with https://cae1.meshcast.io/whep/MyStream
