How to use this demo

  • Pick a room and stream ID. The defaults work locally, or share the same values between devices.
  • Start the publisher to capture your camera, then start the viewer to subscribe to that stream.
  • Toggle the mute button to flip track.enabled; the viewer will receive { videoMuted } payloads instantly.
  • Use “Stop Track” to trigger the ended path. “Acquire New Track” captures a fresh camera and replaces it on the fly.
